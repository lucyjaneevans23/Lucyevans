For this project, I was tasked with creating refreshing a study skills resource for Leeds Beckett University. The goal was to expand the brand's online resources and create a cohesive visual language across digital platforms. The outcomes were a quiz and a infographic with the aim of helping students in their studies. This project was built using educational theory as a basis.

For the motion graphics component of this project, I designed a 60-second animation that explains the topic of Harvard Referencing. The animations features were all designed seperately, by me, and brought together in an animation. The animation is designed to be flexible and can be used as a standalone video or incorporated into social media posts and website intro sequences.

The web design outcome is a responsive quiz tests the user on the information showcased in the infographic. The web component and video were designed to be cohsive with one another and using the existing brand guidelines. The Webpage includes animation, interractive features and delivers an outcome to the user.

Technical Details: HTML, CSS & JavaScript. Adobe Creative Suite (Photoshop, Illustrator, After Effects). Digital Graphics and illustration techniques.
